# awesome-claude-code-toolkit (rohitg00)

A **Claude Code-exclusive** comprehensive toolkit catalog. 1.5k stars, Apache-2.0, actively maintained (459 commits, last update March 2026). Distinct from generalist awesome-lists by providing **reference implementations** rather than just links.

For our wiki, this matters as the **most operationally useful single catalog** — it ships copy-paste-ready agents, hooks, skills, and plugins documented as patterns rather than as "look at this cool repo" links.

The full structure adds up to a substantially-curated reference vs the more aggregator-style awesome-lists.

## How it differs from other catalogs

| Catalog | Style | Best for |
|---|---|---|
| **[[Awesome Claude Code (hesreallyhim)]]** | Aggregator (links) | "Is there a tool for X?" — find existing things |
| **awesome-claude-code-toolkit (rohitg00)** | **Reference implementation** | "Show me a working pattern for X" — adopt directly |
| [[awesome-agent-skills (VoltAgent)]] | Multi-platform skill catalog | Skills curated by major orgs (Anthropic, Microsoft, etc.) |
| [[claude-skills (alirezarezvani)]] | Single-author scale collection | 235+ skills, persona-driven |
| [[agents (wshobson)]] | Single-author multi-tier | 184 agents + 150 skills + 98 commands |

This one's distinctive value: **agent + hook + plugin patterns documented for adoption**. Particularly the 20 hooks and 14 MCP configs — both are categories where most catalogs are thin.

## SkillKit integration

The catalog references **400,000 skills via SkillKit** (https://skillkit.io). Worth noting but skeptically — that's the SkillKit-aggregator number, not curated. The 35 *curated* skills in this catalog are the ones the maintainer vouches for.
